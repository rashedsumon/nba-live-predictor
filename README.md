# NBA Live Game Win-Probability & Lineup Optimizer

An advanced applied machine learning research prototype designed to calculate live game win probabilities, evaluate shifting line-up efficiencies, and recommend optimal player rotations in real-time.

## Features
- **Live Game Context Engine**: Evaluates time, scores, possession, and fouling context.
- **Lineup Assessment**: Uses historical proxy tracking metrics (RAPM/EPM) to evaluate combinations on the court.
- **Predictive Analytics**: Leverages structured mathematical simulations to predict operational outcomes.

## Local Setup
1. Clone this repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `streamlit run streamlit_app.py`
